# shortcut-website

> Generate websites using [Shortcut](https://www.shortcut.com) as a CMS.

## Install

```bash
$ npm install --global shortcut-website
```

## CLI

```
$ shortcut-website --help

  Usage
    $ shortcut-website

  Options
    --name  Site name

  Examples
    $ shortcut-website --name="My Blog"
```
